## 📖 Overview

**Collab.io** is a cutting-edge platform designed to bridge the gap between innovative brands and top-tier creators. Built with **Angular 18** and powered by **Supabase**, this project delivers a premium, Apple-inspired user experience prioritizing beautiful micro-interactions, responsive styling, and fluid animations.

Whether you're a brand looking for the perfect creator for your next campaign, or a creator wanting to find work that aligns with your passions, Collab.io provides the end-to-end tooling you need to communicate, manage campaigns, and execute flawlessly.

## ✨ Key Features

- 🎨 **Apple-Inspired Design System**: A meticulously crafted UI featuring glassmorphism, fluid micro-animations, and a highly polished dark mode.
- 🔐 **Authentication & Authorization**: Secure, robust auth flow powered by Supabase including Login, Signup, and Route Guards.
- 💼 **Comprehensive Dashboard**: A fully-featured workspace tailored for productivity.
  - **Overview**: High-level activity and metrics.
  - **Find Work**: Seamlessly search and discover campaigns matching your skills.
  - **Campaigns**: Create, track, and manage ongoing brand campaigns.
  - **Messages**: In-app communication system.
  - **Profile & Settings**: Manage your brand/creator identity effortlessly.
- ⚡ **High-Performance Architecture**: Leverages Angular 18's latest capabilities coupled with Supabase for real-time data sync and sub-second latency.

## 🛠️ Tech Stack

- **Frontend**: [Angular 18](https://angular.dev/)
- **Backend/Database**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Styling**: Vanilla CSS with modern flexbox/grid layouts and complex animations
- **Routing**: Angular Router with Auth Guards
- **State Management**: RxJS



## 📁 Project Structure

```text
src/
├── app/
│   ├── core/           # Core services (Auth, Supabase, Campaigns) and API logic
│   ├── pages/          # Full-page components (Home, Login, Signup, Dashboard routes)
│   ├── shared/         # Reusable UI components (Navbar, Sidebar), pipes, directives
│   ├── app.routes.ts   # Application routing definitions
│   └── app.component.* # Root component
├── assets/             # Static assets like images and icons
├── environments/       # Environment configuration files
└── index.css           # Global design system, tokens, and utility classes
```
